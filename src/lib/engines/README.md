# Motores de cálculo

Esta pasta destina-se aos motores de cálculo da aplicação:

- **Paramétrico** — geração de peças a partir de módulos e dimensões.
- **Regras de ferragens** — atribuição automática de ferragens por tipo de móvel/junção.
- **Nesting** — otimização do plano de corte de chapas.
- **Exportação MPR** — geração de ficheiros MPR para CNC.

Vazio nesta fase. A fundação da app (auth, modelo de dados, app-shell e definições) está implementada antes.
